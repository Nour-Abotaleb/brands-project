HTML STRUCTURE

The template includes 10 html pages. Every page represents number of features, such as the Home Page which define the website, what the services it represents, the products included, and so on. If you would like to edit the colors, fonts, or styles of any element in this template, you can test the changes by using "Inspect Element" on Chrome, Safari, Mozilla and Opera (press right click on the browser). Change what you want and then add the final code to the template's files. Or change it directly in the template's file by change the hex code of the color or the source of the image..., you can also change any tag in HTML (add or remove), but Don't Forget to change it in CSS and JavaScript files.

CSS FILES AND STRUCTURE

There are 11 CSS files in this template. The first file (home.css) contains all of the styling for the Home Page. The second file (product.css) contains all of the stylings for the Products Page. The third file (documentation.css) contains all of the styling for the Documentation (user manual) Page. The fourth file (contact.css) contains all of the styling for the Help / Contact Page. and CSS files for styling the shopping cart page and the checkout page. The last file contains all the fonts used in the template (fonts.css). Every file is separated into sections and well-documented.

JAVASCRIPT FILE

The template includes 1 JAVASCRIPT file for all the html files. It has all the functionality included in the website, such as add or remove product from the cart. Any action happens in the template is made by JAVASCRIPT code.

Attention: If you change or remove any of JavaScript's code, your site will not work properly. So be careful if you don't know how to edit the code, you can ask for help from an expert.
